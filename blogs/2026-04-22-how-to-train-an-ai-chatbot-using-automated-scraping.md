You&#x2019;ll access Apify Console, a workspace for running and building web automation tools.</p><figure class="kg-card kg-image-card"><a href="https://console.apify.com/actors/aYG0l9s7dbB7j3gbS?addFromActorId=aYG0l9s7dbB7j3gbS"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="1152" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/WebContentCrawler_SignUp.png" width="1826" /></a></figure><p>Website Content Crawler can render dynamic content and extract the meaningful text while removing navigation elements, ads, and other noise.</p><h3 id="step-2-configure-the-scraper-and-run-it">Step 2: Configure the scraper and run it</h3><p>In this tutorial, we&#x2019;ll extract data that a travel company needs to launch a chatbot that helps users with questions about their flights - baggage rules, refund policies, or visa requirements. To answer these questions accurately, the chatbot needs access to <strong>reliable travel information</strong> from websites such as airline help centers. We&#x2019;ll use <a href="https://help.ryanair.com/"><code>https://help.ryanair.com</code></a> as our <strong>Start URL</strong>.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="1596" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/WebsiteContentCrawler_UI.png" width="1428" /></figure><p>You can also force the crawler to skip certain URLs using the&#xa0;<strong>Exclude URLs (globs)</strong>&#xa0;input setting, which specifies an array of glob patterns matching URLs of pages to be skipped. Note that this setting affects only links found on pages, but not&#xa0;<strong>Start URLs</strong>, which are always crawled.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="530" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/WebsiteContentCrawler_Exclude.png" width="1130" /></figure><p>You can also customize your crawl further and select your output type. We&#x2019;ll select the Markdown toggle as it will keep the content clean, structured, and easy for AI models to interpret.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="560" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/MarkdownOutput.png" width="1108" /></figure><p>Under the <strong>Browser behavior</strong> setting, you can select elements to exclude from the final results, such as cookie banners and navigation menus.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="916" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/BrowserBehavior.png" width="1100" /></figure><p>Using the <strong>Crawler identification</strong> option, you can set up a proxy to access any website. If your website is Cloudflare-protected, you can set up Signed HTTP requests. To do this, go to the <a href="https://radar.cloudflare.com/bots/directory/apify-website-content-crawler">Cloudflare bot directory</a> to create credentials, then paste them into the Custom HTTP headers setting. Keep the Sign HTTP requests toggle on.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="322" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/WebsiteContentCrawler_CustomHTTPHeaders.png" width="1090" /></figure><p>To keep your scraping costs predictable, you can set up a maximum cost per run under the <strong>Run options</strong><em>.</em></p><p>Once you&#x2019;re happy with your choices, click&#xa0;<strong>Save &amp; Start</strong>. After a couple of minutes, the run will finish, and you&#x2019;ll be able to check the results in the preview table, in clean Markdown:</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="841" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/PreviewTable_WebsiteContentCrawler.png" width="2000" /></figure><p>You can also download the results as JSON, Excel, CSV, and more by clicking the <strong>Export</strong> button. </p><div class="kg-card kg-callout-card kg-callout-card-blue"><div class="kg-callout-emoji">&#x1f4a1;</div><div class="kg-callout-text">The <a href="https://docs.apify.com/platform/integrations/pinecone">Apify-Pinecone integration</a> automatically sends crawled website content to a vector database so AI systems can retrieve it as context when generating answers.</div></div><h3 id="step-3-schedule-automated-runs">Step 3: Schedule automated runs</h3><p>If you want to scrape data for your AI chatbot regularly, you can schedule Apify scrapers to run automatically and collect data without manual input.</p><p>First, make sure your scraper is properly configured, then click the&#xa0;<strong>Save as a new task</strong>&#xa0;button in the top-right corner. </p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="622" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/SaveAsANewTaskButton.png" width="1932" /></figure><p>Give your task a name and save it. Next, you can easily schedule the task by accessing&#xa0;<strong>Schedules</strong>&#xa0;in the left-hand navigation and clicking the&#xa0;<strong>Create a schedule</strong>&#xa0;button:</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="441" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/CreateSchedule-1.png" width="884" /></figure><p>We&#x2019;ve already saved our task, so now it&#x2019;s time to add it to the schedule. Pick the foundation model your chatbot runs on - OpenAI, Anthropic, Gemini, or any other model.</li><li>Choose a <a href="https://blog.apify.com/what-is-a-vector-database/">vector database</a> (such as Pinecone). Vector databases store your content as embeddings - numerical representations of text that let the system find semantically similar information fast. Website Content Crawler outputs clean Markdown that&apos;s ready for embedding.</li><li>Connect everything together. This is where you wire crawled content into your vector database and LLM, so the chatbot can actually retrieve and use it. There are several ways to do this, depending on your stack and the level of control you need. Examples include:<ul><li><a href="https://blog.apify.com/how-to-use-langchain/">LangChain</a> orchestration</li><li>Apify API to fetch the results and push them into your own pipeline</li><li>visual workflow automations such as n8n or Make</li></ul></li></ol><div class="kg-card kg-callout-card kg-callout-card-blue"><div class="kg-callout-emoji">&#x1f4a1;</div><div class="kg-callout-text">Try the dedicated <a href="https://n8n.io/workflows/14157-build-a-company-website-rag-chatbot-using-apify-pinecone-and-gemini/">Apify n8n template to connect your AI chatbot to data</a>. n8n gives you full access to code nodes and custom logic when you need it. With n8n workflow data ingestion, indexing, and answering queries happen automatically and stay up to date as your source content changes.</div></div><h3 id="step-5-retrieving-real-time-travel-information">Step 5: Retrieving real-time travel information</h3><p>Now that you already have a knowledge base and a workflow set up, it&#x2019;s time to prepare your chatbot for cases where users ask questions <strong>outside of the indexed dataset</strong>. In those cases, the chatbot will retrieve information directly from the web.</p><p><a href="https://console.apify.com/organization/h95HnAFn4jqg8ZkTR/actors/3ox4R101TgZz67sLr/input">RAG Web Browser</a> enables this capability. It is optimized for speed. When used in standby mode, you can get responses within 16 seconds.</p><p>It provides web browsing functionality for AI agents and LLM applications, similar to the&#xa0;<a href="https://openai.com/index/introducing-chatgpt-search/">web browsing</a>&#xa0;feature in ChatGPT, and has been designed for easy integration with LLM applications, GPTs, OpenAI Assistants, and RAG pipelines using function calling.</p><p>This Actor accepts a search phrase or a URL as input, queries Google Search, then crawls web pages from the top search results, cleans the HTML, converts it to text or Markdown, and returns it for processing by the LLM application. We&#x2019;ll demo how it retrieves information using the following search term: <code>visa requirements for Japan EU citizens</code></p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="1046" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/RAGWebBrowser_UI.png" width="1492" /></figure><p>We also set the maximum number of top organic Google Search results whose web pages will be extracted to 3.</p><p>After the Actor finishes running, we can see that it retrieved relevant sources such as the Ministry of Foreign Affairs of Japan and the Embassy of Japan in the US, providing reliable information that an AI chatbot could use to generate an answer.</p><figure class="kg-card kg-image-card"><img alt="How to train an AI chatbot using automated scraping" class="kg-image" height="676" src="https://storage.ghost.io/c/f2/6e/f26ec999-9a90-4aee-a0d4-9b3ca2bb668f/content/images/2026/03/RAGWebBrower_ExampleOutput.png" width="2000" /></figure><div class="kg-card kg-callout-card kg-callout-card-blue"><div class="kg-callout-emoji">&#x1f4a1;</div><div class="kg-callout-text">You can use the <i><em class="italic" style="white-space: pre-wrap;">site</em></i> parameter to exclude specific websites from results. For example, <code style="white-space: pre-wrap;">-site:apify.com RAG</code> will give you results about RAG from websites that aren&apos;t Apify. This is especially useful if you&#x2019;ve already used RAG to embed Apify website data in your chatbot.</div></div><h2 id="conclusion">Conclusion</h2><p>A chatbot is only useful if it can actually answer questions correctly. With automated scraping handling the heavy lifting, you can focus on building the chatbot itself rather than wrangling data.</p><p>Once the scraping pipeline is running on a schedule and RAG Web Browser is handling the gaps, your chatbot has what it needs to stay accurate without any manual intervention.</p>
